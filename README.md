# Azure Banking Data Platform

A personal project simulating a mid-size bank's Azure data platform.
Built to consolidate transaction and reporting data from multiple source 
systems into a centralized Azure solution.

## Architecture (In Progress)
- ADLS Gen2 — central data lake storage
- Azure Data Factory — pipeline orchestration  
- Azure Databricks + PySpark — ETL processing
- Azure SQL Database — reporting layer
- Azure Monitor + KQL — observability
- Key Vault + Managed Identities + RBAC — security
